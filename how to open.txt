To open the .jar file.
Step 1.) Open the Command Prompt
	Open the Command Prompt window by clicking the Start button Picture of the Start button, clicking All Programs, 
	clicking Accessories, and then clicking Command Prompt.
	Another way to open a Command Prompt window: Click the Start button. 
	In the Search box, type Command Prompt, and then, in the list of results, double-click Command Prompt.

Step 2.) Input into the Command Prompt 
	java -jar "\path\to\YOUR_JAR_NAME.jar"
